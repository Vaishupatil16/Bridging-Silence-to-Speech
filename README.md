

# BRIDGING SILENCE TO SPEECH

A real-time Indian Sign Language recognition system that converts hand gestures into text and speech using computer vision and machine learning.

## FEATURES
- Real-time hand gesture detection using webcam
- Indian Sign Language recognition
- Text-to-speech conversion
- Built using Python, OpenCV, MediaPipe

## HOW TO RUN
1. Create virtual environment
2. Install requirements
3. Train model
4. Run camera inference

## DATASET
Indian Sign Language Dataset (Kaggle)
